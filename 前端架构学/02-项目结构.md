# 2 项目结构分析
```
看了一圈之后，你会发现，一般会有这些目录：

config 配置文件

assets 放静态资源的，图片等
locales 本地化

libs js库/工具包
plugins 插件库/工具包
utils 工具包

views 视图 可能放vue当页面组件/html文件/React组件
pages 同上

layouts 布局 一般也是组件

components 放组件的
templates 模板

models 状态管理/数据管理/数据流/数据模型 vuex/redux的模块

services 服务层，一般放的是写有具体请求后台的文件

dist 打包结果
static 可能是打包结果 也可能是静态资源

mock 模拟数据

e2e 测试
test 测试

```
